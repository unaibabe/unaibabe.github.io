Welcome to the repository of my personal portfolio website! This project is a premium, high-fidelity single-page web application that showcases my unique professional background: bridging **5+ years of AAA Hollywood-grade VFX simulation pipelines** (at El Ranchito, working on Oscar-nominated films) with **academic and hands-on Backend & Full-Stack Software Development**.

## рџ’» How to Run Locally

Since this is a lightweight frontend application optimized for speeds, it requires **zero dependencies** or complex builds.

### Option 1: Direct Open
1. Clone the repository.
2. Double-click the `index.html` file to open it in any modern browser.

### Option 2: Local Server (Recommended for perfect asset serving)
Run a lightweight HTTP server inside the folder using your preferred tool:
* **Python**: `python -m http.server 8000` (then open `http://localhost:8000`)
* **Node.js**: `npx serve` or using the VSCode **Live Server** extension.
